# ZENITH-OKR-DESIGN

## Quando usar
- OKR program from scratch
- OKR refresh (não estão a funcionar)
- Cascading (company → team → individual)
- Quarterly check-ins design
- Year-end retrospective

## Frameworks
- **OKR (Doerr "Measure What Matters"):** O qualitativo + 3-5 KRs quantitativos
- **V2MOM (Salesforce):** Vision/Values/Methods/Obstacles/Measures
- **RPM (Wodtke "Radical Focus"):** team-level OKRs + weekly check-ins
- **Hoshin Kanri (Lean):** strategy deployment
- **NCT (Ravi Mehta):** Narrative/Commitments/Tasks

## Princípios chave
- **Ambitious:** 70% achievement = success ("moonshot")
- **Qualitative O + Quantitative KR**
- **Bottom-up + top-down:** alinhar, não cascatear cego
- **Public:** transparência cross-team
- **Decoupled from compensation:** senão = sandbag

## Templates
1. OKR template (O + 3-5 KRs + initiatives)
2. OKR scoring (0-1.0 scale)
3. Weekly check-in (RPM-style)
4. Quarterly grading + retro
5. Cascading workshop (top → teams)
6. Anti-pattern detector (sandbagged? task-list disfarçada?)
