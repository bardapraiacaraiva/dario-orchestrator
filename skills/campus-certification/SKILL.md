# CAMPUS-CERTIFICATION

## Quando usar
- Setup digital credentials (badges, certificates)
- Diploma digital (registro acadêmico nacional)
- Blockchain-verified credentials
- Stackable credentials program
- Corporate certification (sales, technical)

## Padrões
- **IMS Open Badges 3.0:** W3C Verifiable Credentials based
- **W3C Verifiable Credentials:** standard cross-domain
- **Blockcerts (MIT):** blockchain-anchored
- **EBSI (EU):** European Blockchain Services Infra
- **Diploma Digital (MEC):** XML-padronizado MEC

## Tipos
- **Achievement badge:** completou curso
- **Skill badge:** demonstrou habilidade específica
- **Participation badge:** atendeu evento
- **Milestone:** progressão (e.g., 50% concluído)
- **Stackable credentials:** N badges = certificate

## Templates
1. Badge taxonomy (níveis + categorias)
2. IMS Open Badges 3.0 emission API
3. Diploma digital MEC XML
4. Blockchain credential strategy
5. Verification portal público
