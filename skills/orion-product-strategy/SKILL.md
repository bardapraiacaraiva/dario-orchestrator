# ORION-PRODUCT-STRATEGY

## Quando usar
- Definir vision/mission de produto (greenfield ou re-platforming)
- Criar North Star Metric (NSM) e supporting metrics
- Strategic narrative para investors/board
- Annual planning ciclo (OKRs anuais)
- Pivot/strategic re-direction

## Frameworks
- **Good Strategy Bad Strategy (Rumelt):** kernel = diagnosis + guiding policy + actions
- **Playing to Win (Lafley):** where to play / how to win / capabilities
- **Working Backwards (Amazon):** PR/FAQ → spec → build
- **3 Horizons (McKinsey):** core / adjacent / transformational
- **Wardley Maps:** componentes + evolution stage
- **JTBD (Christensen):** outcome-driven

## Templates
1. Vision + mission + values document
2. North Star Metric tree (NSM + 3-5 input metrics)
3. Strategic narrative deck (15 slides)
4. PR/FAQ Amazon-style
5. OKR quarterly cascade (company → product → team)
