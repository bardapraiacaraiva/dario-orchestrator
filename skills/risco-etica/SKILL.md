# RISCO-ETICA: Ethics & Code of Conduct Skill

## When to Activate

**Trigger words (PT):** codigo de conduta, etica, conflito de interesses, ofertas e hospitalidades, politica de etica, declaracao de interesses, integridade, compliance, anti-corrupcao, formacao etica
**Trigger words (EN):** code of conduct, ethics, conflict of interest, coi, gifts policy, hospitality, integrity, anti-corruption, anti-bribery, ethics training, tone at the top, compliance culture

## Step-by-Step Workflow

### Phase 1: Code of Conduct Drafting
1. Define organizational values and principles
2. Structure sections: purpose, scope, core values, expected behaviors
3. Cover key topics: anti-corruption, COI, gifts, confidentiality, discrimination, data protection, environmental responsibility
4. Write in clear, accessible language (avoid legalistic jargon)
5. Translate to all relevant languages (minimum: PT and EN)
6. Include reporting mechanisms and non-retaliation clause
7. Obtain Board approval and CEO endorsement letter

### Phase 2: Conflict of Interest (COI)
1. Define COI categories: financial, personal, professional, family
2. Design annual COI declaration form
3. Establish disclosure process (who, when, how)
4. Create COI review committee or procedure
5. Document management measures (recusal, divestiture, disclosure)
6. Maintain COI register with periodic updates

### Phase 3: Gifts & Hospitality Policy
1. Define thresholds (e.g., EUR 50 per gift, EUR 150 per hospitality event)
2. Establish pre-approval process for above-threshold items
3. Create gifts register (given and received)
4. Prohibit: cash gifts, gifts to/from public officials (unless de minimis), gifts during procurement
5. Define exceptions (corporate branded items, industry events)
6. Quarterly review of gifts register for patterns

### Phase 4: Anti-Corruption
1. Assess corruption risk by geography, sector, and counterparty
2. Map touch points with public officials
3. Implement facilitation payments prohibition
4. Third-party due diligence for agents and intermediaries
5. Charitable donations and sponsorship approval process
6. Align with PT Penal Code (Art. 372-374) and UK Bribery Act / FCPA if applicable

### Phase 5: Training & Communication
1. Annual mandatory ethics training for all employees
2. Enhanced training for high-risk roles (procurement, sales, finance)
3. New hire onboarding includes Code of Conduct acknowledgment
4. Periodic communications: case studies, Q&A, ethics reminders
5. Track completion rates and test comprehension

### Phase 6: Monitoring & Enforcement
1. Ethics metrics: declarations filed, reports received, investigations closed
2. Disciplinary framework for violations (proportional, consistent)
3. Periodic reviews of Code effectiveness
4. Annual ethics risk assessment
5. Board/Audit Committee reporting on ethics program

## Red Flags

- No Code of Conduct or code not updated in >3 years
- COI declarations not collected annually
- No gifts register or threshold enforcement
- Ethics training not mandatory or completion below 80%
- No confidential reporting channel for ethics concerns
- Retaliation against reporters documented or suspected
- Anti-corruption risk assessment never conducted
- No senior management "tone at the top" communication
- Third-party intermediaries without due diligence
- Donations/sponsorships without approval process
- Disciplinary inconsistency (different treatment for similar violations)
- Ethics program exists only on paper (no active monitoring)
