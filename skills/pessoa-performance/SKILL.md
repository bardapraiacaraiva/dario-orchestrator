## Workflow

### Step 1: Performance Cycle Design

| Element | Option A: Traditional | Option B: Continuous | Option C: Hybrid |
|---------|---------------------|---------------------|-----------------|
| **Goal setting** | Annual (January) | Quarterly OKRs | Quarterly OKRs |
| **Check-ins** | Mid-year + Annual | Weekly 1:1s | Bi-weekly 1:1s |
| **Formal review** | Annual (December) | Quarterly review | Semi-annual |
| **360 feedback** | Annual | Quarterly pulse | Semi-annual |
| **Calibration** | Annual | Quarterly | Semi-annual |
| **Compensation link** | Annual merit | Quarterly bonus | Annual merit + quarterly bonus |

**Recommended**: Option C (Hybrid) — Quarterly OKRs with semi-annual reviews.

### Step 2: OKR Design (Individual Level)

**OKR Template:**

| Element | Content |
|---------|---------|
| **Objective** | Qualitative, inspiring, ambitious (what you want to achieve) |
| **Key Result 1** | Quantitative, measurable, time-bound (how you know you achieved it) |
| **Key Result 2** | Quantitative, measurable, time-bound |
| **Key Result 3** | Quantitative, measurable, time-bound |
| **Alignment** | Links to team/company OKR: [which one] |
| **Owner** | [Employee name] |
| **Quarter** | Q[X] [YEAR] |

**OKR Scoring (Doerr):**
- 0.0-0.3: Failed to make progress
- 0.4-0.6: Made progress but fell short
- 0.7-1.0: Delivered (0.7 is the target for stretch OKRs)

**Rules:**
- 3-5 OKRs per person per quarter (not more)
- 60% committed OKRs (must hit), 40% aspirational (stretch)
- OKRs are NOT tied directly to compensation (promotes sandbagging)
- Review and score at end of quarter, set new ones

### Step 3: 1:1 Meeting Framework (Grove/Scott)

**Weekly/Bi-weekly 1:1 Agenda (30-45 min):**

| Block | Time | Content |
|-------|------|---------|
| **Check-in** | 5 min | "How are you? What's on your mind?" |
| **Progress** | 10 min | OKR/goal progress update, blockers |
| **Feedback** | 10 min | Bi-directional (manager gives AND receives) |
| **Development** | 5 min | Skills, career, learning goals |
| **Action items** | 5 min | Clear next steps with owners |

**Manager Preparation Checklist:**
- [ ] Reviewed their OKR dashboard
- [ ] Noted specific examples of good/improvable work
- [ ] Prepared one piece of developmental feedback
- [ ] Asked for upward feedback ("What can I do better?")

### Step 4: 360 Feedback Design

**Raters:**

| Rater Type | Number | Weight | What They Assess |
|-----------|--------|--------|-----------------|
| **Self** | 1 | 10% | Self-awareness, reflection |
| **Manager** | 1 | 30% | Results, competencies, growth |
| **Peers** | 2-4 | 30% | Collaboration, communication, reliability |
| **Direct reports** | 2-4 | 20% | Leadership, support, clarity |
| **External** (optional) | 1-2 | 10% | Client/partner perspective |

**360 Competency Dimensions:**

| Competency | Description | Rating (1-5) |
|-----------|-------------|-------------|
| **Delivers results** | Consistently achieves goals and commitments | /5 |
| **Collaboration** | Works effectively across teams and functions | /5 |
| **Communication** | Clear, timely, appropriate communication | /5 |
| **Innovation** | Proposes and implements improvements | /5 |
| **Leadership** | Inspires, develops, and empowers others | /5 |
| **Culture** | Lives company values, positive influence | /5 |

**360 Open-Ended Questions:**
1. "What should this person CONTINUE doing?"
2. "What should this person START doing?"
3. "What should this person STOP doing?"

### Step 5: Performance Rating Scale

| Rating | Label | Description | Expected Distribution |
|--------|-------|-------------|---------------------|
| 5 | **Exceptional** | Consistently exceeds all expectations, role model | 5-10% |
| 4 | **Exceeds** | Frequently exceeds expectations in key areas | 20-25% |
| 3 | **Meets** | Fully meets all expectations, solid performer | 50-60% |
| 2 | **Developing** | Partially meets expectations, needs improvement | 10-15% |
| 1 | **Below** | Does not meet expectations, urgent action needed | 0-5% |

### Step 6: Calibration Session

**Purpose**: Ensure fair, consistent ratings across managers and teams.

**Process:**
1. All managers submit preliminary ratings
2. HR compiles distribution analysis
3. Calibration meeting (all managers + HR, 2-4 hours)
4. Review each employee rated 5 or 1 (justify with evidence)
5. Discuss borderline 3/4 and 2/3 cases
6. Adjust for manager leniency/severity bias
7. Finalize ratings

**Calibration Rules:**
- Every rating must be supported by specific examples
- No forced distribution, but flag outlier departments
- Address unconscious bias (gender, recency, halo effect)
- Document all calibration decisions

### Step 7: Performance Improvement Plan (PIP)

**PIP Template:**

| Element | Detail |
|---------|--------|
| **Employee** | [Name] |
| **Manager** | [Name] |
| **Start date** | [Date] |
| **Duration** | 30 / 60 / 90 days |
| **Performance gap** | [Specific, documented shortfall] |
| **Expected standard** | [What "meets expectations" looks like] |
| **Actions required** | [Specific steps the employee must take] |
| **Support provided** | [Training, coaching, resources offered] |
| **Check-in cadence** | Weekly meetings with documented progress |
| **Success criteria** | [Measurable outcomes by end of PIP] |
| **Consequence** | [What happens if PIP not met: role change, dismissal] |

**PT Legal Note**: Document everything. Under Codigo do Trabalho, dismissal for poor performance requires documented evidence of underperformance, training/support provided, and formal warnings.

### Step 8: Promotion Framework

**Promotion Criteria Matrix:**

| Criterion | Weight | Evidence Required |
|-----------|--------|-----------------|
| **Sustained performance** | 30% | 2+ consecutive "Exceeds" or "Exceptional" ratings |
| **Next-level competencies** | 25% | Already demonstrating behaviors of target grade |
| **Impact beyond role** | 20% | Cross-team contributions, mentoring, projects |
| **Skills readiness** | 15% | Technical/functional skills for next level |
| **Business need** | 10% | Role/headcount exists at next level |

**Promotion Process:**
1. Manager nominates (with evidence package)
2. HR reviews against criteria matrix
3. Calibration committee approves/defers
4. Compensation adjustment calculated (typically 8-15% increase)
5. Communication to employee with new expectations
6. 90-day review at new level

## Red Flags

Stop and warn the user if:
- No documented performance criteria before evaluation period
- Ratings based on recency bias (last 2 weeks, not full period)
- Manager has never given feedback before the formal review (ambush)
- 100% of team rated "Exceeds" or "Exceptional" (grade inflation)
- PIP used as a surprise with no prior feedback or support
- Promotion decisions based on tenure alone, not competency
- No calibration process (inconsistent standards across managers)
- Performance system used punitively instead of developmentally
- OKRs set by manager without employee input
- No documentation of underperformance before considering dismissal (PT legal risk)
- 360 feedback used as anonymous weapon instead of development tool
