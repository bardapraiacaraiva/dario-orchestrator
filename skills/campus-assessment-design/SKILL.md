# CAMPUS-ASSESSMENT-DESIGN

## Tipos
- **Diagnóstica:** baseline antes de instrução
- **Formativa:** checks during learning (low-stakes)
- **Sumativa:** final-of-unit (graded)
- **Autêntica:** real-world task simulation
- **Adaptiva:** dificuldade ajusta a performance

## Frameworks
- **Bloom's Taxonomy (revisada):** níveis cognitivos
- **Webb's DOK:** 4 levels — recall, skill, strategic, extended thinking
- **SOLO Taxonomy:** structure of learning outcomes
- **Backward Design:** assessment como evidência

## Templates
1. Test blueprint (DOK × content)
2. Rubric design (4 levels, observable criteria)
3. Item bank com TRI calibration (ENEM-style)
4. Performance task design (autêntica)
5. Peer assessment + self-assessment templates

## Princípios
- **Validity:** mede o que diz medir
- **Reliability:** consistent across raters
- **Fairness:** sem viés cultural/linguístico
- **Transparency:** rubric shared upfront
- **Actionable feedback:** beyond letter grade
