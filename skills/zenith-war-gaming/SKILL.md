# ZENITH-WAR-GAMING

## Quando usar
- Major launch (antecipar competitor response)
- New market entry (incumbents react how?)
- M&A defense (hostile bid)
- Crisis simulation
- Pricing change (competitor matching?)

## Setup
- **Players:** 3-5 teams (us + 2-4 competitors + regulators/market)
- **Format:** 2-3 day workshop ou multi-week
- **Moves:** rounds (months or quarters)
- **Umpire:** judge moves, inject realism
- **Debrief:** lessons, vulnerabilities, strategic implications

## Templates
1. War game design doc (objective + players + rules)
2. Competitor profile per team (CEO archetype + capabilities)
3. Move template (action + rationale + expected response)
4. Umpire scoring rubric
5. Post-game debrief (what we learned about ourselves + market)
