# GAIA-SOCIAL-IMPACT

## Quando usar
- Social enterprise impact reporting
- Foundation/philanthropy impact measurement
- Impact investing portfolio reporting
- B-Corp certification preparation
- ESRS S3 (Affected communities) disclosure
- Public sector program evaluation

## Frameworks
- **Theory of Change (TOC):** inputs → activities → outputs → outcomes → impact
- **SROI (Social Return on Investment):** £ value per £ invested
- **IRIS+ (Impact Frontiers):** standardized impact metrics
- **GRI 200-400 series:** social performance
- **SDG indicators:** UN Sustainable Development Goals alignment
- **OECD Social Impact Measurement Framework**

## SROI workflow
```
1. Stakeholder identification
2. Outcomes mapping
3. Evidence of outcomes (surveys, data)
4. Financial proxies (£ value of outcomes)
5. Attribution + deadweight + drop-off
6. Calculate SROI ratio (typically 1:3 to 1:8 for good programs)
7. Sensitivity analysis
8. Report to stakeholders
```
