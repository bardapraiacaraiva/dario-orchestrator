# OBSIDIAN-SECOND-BRAIN

## Filosofia (Tiago Forte)
**Your brain is for having ideas, not storing them.** Externalize, organize, retrieve.

## CODE Method
1. **Capture:** salve apenas o que ressoa (não tudo)
2. **Organize:** PARA (Projects/Areas/Resources/Archive)
3. **Distill:** progressive summarization (highlights → bold → 1-liner)
4. **Express:** turn knowledge into output

## Quando usar
- Setup de Second Brain individual (greenfield)
- Migration Notion → Obsidian
- Audit de PKM existente (capture too much? express too little?)
- Capture pipeline (Readwise + email + voice)

## Stack
- **Obsidian** (local-first, markdown)
- **Notion** (collaborative)
- **Roam Research** (bidirectional + queries)
- **Logseq** (open-source, block-based)
- **Tana** (super-tags)
