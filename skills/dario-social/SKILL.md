### 3. Post Copy Generator (per platform)

**Instagram Feed:**
- Hook (primeira linha — para no scroll): 7-10 palavras, forte
- Body: 3-5 linhas, valor ou storytelling
- CTA: 1 linha clara (comenta, guarda, partilha, link in bio)
- Hashtags: 10-15 (mix: 5 broad + 5 niche + 3 branded + 2 location)
- Max 2200 chars

**Instagram Reels (script):**
```
HOOK (0-3s): [visual forte + texto on-screen]
SETUP (3-8s): [contexto rapido]
VALUE (8-25s): [o conteudo principal — tip, reveal, demo]
CTA (25-30s): [segue, guarda, comenta]
AUDIO: [trending audio ou original]
TEXT OVERLAY: [resumo em 3-5 bullet points]
```

**LinkedIn Post:**
- Hook: primeira frase forte (aparece antes do "ver mais")
- Structure: hook → context → insight → proof → CTA
- Tone: profissional mas humano, nao corporativo
- Hashtags: 3-5 max (LinkedIn penaliza excesso)
- Formatting: paragrafos curtos, espacamento, emojis moderados
- Best length: 1200-1500 chars

**TikTok Script:**
```
HOOK (0-2s): [frase + visual que para o scroll]
CONTENT (2-45s): [rapido, directo, visual-first]
CTA (ultimo 3s): [segue para parte 2 / comenta / partilha]
SOUND: [trending ou original]
TEXT: [grande, centrado, alto contraste]
HASHTAGS: 3-5 max
```

### 5. Community Management Templates

**Resposta a comentario positivo:**
```
Obrigado/a, [nome]! Ficamos muito contentes que [referencia ao comentario]. [Pergunta para continuar conversa] 💬
```

**Resposta a pergunta:**
```
Ola [nome]! [Resposta directa]. Se precisar de mais info, envie-nos DM — temos todo o gosto em ajudar! 📩
```

**Resposta a critica:**
```
[Nome], lamentamos que a sua experiencia nao tenha sido a melhor. Queremos resolver — pode enviar-nos DM com os detalhes? Vamos tratar disto pessoalmente. 🙏
```

**Resposta a spam/troll:**
```
[Ocultar comentario + bloquear se reincidente. Nao responder publicamente.]
```

### 6. Content Pillars Framework

Para cada cliente, definir 4-5 pilares:

| Pilar | % do conteudo | Objectivo | Exemplo (restaurante) |
|---|---|---|---|
| **Educativo** | 30% | Autoridade | "3 sinais de que o peixe e fresco" |
| **Entertainment** | 25% | Alcance | Chef a grelhar em slow-mo |
| **Behind scenes** | 20% | Conexao | "O que chegou da lota hoje" |
| **Social proof** | 15% | Confianca | Reviews, UGC, press mentions |
| **Promocional** | 10% | Conversao | Menu especial, reservas, eventos |

**Regra 80/20:** 80% valor (educativo + entertainment + BTS) / 20% venda (proof + promo)

## Red Flags

- NUNCA postar sem aprovacao do cliente (calendario e proposta, nao publicacao)
- Respeitar ratio 80/20 — excesso de promo = unfollow
- Hashtags: Instagram max 15, LinkedIn max 5, TikTok max 5
- Reels/TikTok: primeiros 2 segundos decidem tudo — investir no hook
- Community management: responder em <4h (horario util) — nunca ignorar
