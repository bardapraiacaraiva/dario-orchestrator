# HELIOS-GRID-INTEGRATION

## Workflow GD interconnect BR
```
1. Estudo preliminar (consumidor)
2. Solicitação à distribuidora (formulário)
3. Parecer técnico (15-30 dias distribuidora)
4. Aprovação projeto + orçamento conexão
5. Construção (consumidor + distribuidora)
6. Vistoria + comissionamento
7. Energização + medição bidirecional
8. SCEE registro (compensação)
```

## PRODIST relevant modules
- **Módulo 3:** acesso ao sistema
- **Módulo 7:** cálculo de perdas
- **Módulo 8:** qualidade da energia
- **Módulo 9:** medição
