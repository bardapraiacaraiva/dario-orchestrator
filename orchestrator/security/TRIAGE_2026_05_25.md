- **pip 25.3 ‚Üí 26.1** ‚Äî out-of-band, just `pip install -U pip` in venv. CI already pins via `setup-python` cache so it'll catch up on next CI run.

### Major version (deferred, needs integration testing)

### Track upstream (no fix available)
- **diskcache 5.6.3 CVE-2025-69872** ‚Äî no patched version exists. Need to:
