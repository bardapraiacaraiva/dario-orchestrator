## Installation

```bash
# Clone
git clone https://github.com/bardapraiacaraiva/dario-orchestrator.git ~/.claude-orchestrator

# Run installer (creates ~/.claude/orchestrator/ structure)
bash ~/.claude-orchestrator/orchestrator/install.sh

# Or copy skills directly
cp -r ~/.claude-orchestrator/skills/* ~/.claude/skills/
```

## Requirements

- [Claude Code CLI](https://claude.ai/code) installed
- macOS, Linux, or Windows (Git Bash/MSYS2)

## Architecture

```
User Request
    |
    v
[CEO - Orchestrator]
    |
    +-- UNDERSTAND -- Parse intent, gather context
    +-- DECOMPOSE  -- Break into atomic tasks
    +-- DISPATCH   -- Route to right agent/skill
    +-- EXECUTE    -- Workers run in heartbeat windows
    +-- REVIEW     -- Execution policies enforced
    +-- SYNTHESIZE -- Combine outputs
    +-- AUDIT      -- Log everything, update budget
```

## Files NOT in this repo (generated per install)

- `company.yaml` — Your org hierarchy (generated by `install.sh`)
- `tasks/active/` — Active task files
- `tasks/done/` — Completed tasks
- `budgets/` — Monthly budget tracking
- `audit/` — Audit trail logs
